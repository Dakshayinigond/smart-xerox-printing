Smart Xerox Printing System
## Project Overview

Smart Xerox Printing System is a web-based application developed to manage and automate printing services.
The system allows users to upload documents, select print options, calculate cost, and manage print requests efficiently.

This project is developed as part of an academic mini project to demonstrate full-stack web application development using modern frontend and backend technologies.

## Features
- User Registration and Login
- Upload documents for printing
- Select print options (pages, copies, color, etc.)
- Calculate printing cost
- Order and request management
- Admin dashboard for managing print jobs
- Responsive UI
